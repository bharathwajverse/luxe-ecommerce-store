# Contributing to FlipShop

Thanks for your interest in contributing to FlipShop! Your improvements help make this e-commerce project stronger and easier to use.

## How to contribute

1. Fork the repository.
2. Create a new branch with a descriptive name, e.g. `feature/add-product-filters` or `fix/cart-bug`.
3. Install dependencies:

```bash
npm install
```

4. Make your changes.
5. Run the development server to verify your updates:

```bash
npm run dev
```

6. Submit a pull request with a clear description of the changes.

## Reporting issues

If you find a bug or want to request a new feature, open an issue with:

- A descriptive title
- Steps to reproduce the problem
- Expected vs actual behavior
- Screenshots or code snippets if helpful

## Code style

- Follow the existing React and JavaScript patterns used in the project.
- Keep components reusable and well-structured.
- Use the existing `src/styles/index.css` file for global styles.

## Notes

- This repo uses Vite for development and build tooling.
- Existing scripts are available in `package.json`.
- No automated test suite is currently included.
