# Security Policy

## Reporting a Vulnerability

If you discover a security issue in FlipShop, please report it responsibly by opening an issue and marking it as a security concern, or contact the repository maintainers directly.

## Supported Versions

This repository is currently maintained in its latest version only. Security fixes will be applied to the current release branch.

## Disclosure Policy

Please do not disclose security vulnerabilities publicly until a fix has been released.

## Contact

If you need to report a security issue privately, please use the repository issue tracker or contact the maintainers through GitHub.
